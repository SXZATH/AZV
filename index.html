AZV Suite Details

AZV es un sistema de gestión de paquetes para Bash que simplifica la instalación de librerías y el scripting.

  • Se instala y configura automáticamente para reducir la intervención manual.
  • Los módulos son importados a scripts con 'source utils' para usar las funciones o variables dentro.
  • Comandos de AZV:
  • 'azv --help' - Proporciona ayuda en la herramienta.
  • 'azv list' - Lista todos los módulos instalados en el sistema.
  • 'azv install' - Instala módulos como vulnhammer para seguridad SSH.
  • 'azv remove' - Remueve los módulos seleccionados.
