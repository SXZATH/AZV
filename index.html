AZV Suite Details

AZV es un sistema de gestión de paquetes para Bash que simplifica la instalación de librerías y el scripting.

  • Se instala y configura automáticamente para reducir la intervención manual.
  • Los módulos son importados a scripts con 'source utils' para usar las funciones o variables dentro.
  • Comandos de AZV:
  • 'azv --help' - Proporciona ayuda en la herramienta.
  • 'azv list' - Lista todos los módulos instalados en el sistema.
  • 'azv install' - Instala módulos como vulnhammer para seguridad SSH.
  • 'azv remove' - Remueve los módulos seleccionados.

Hydra Details

Hydra es una herramienta de ataques de fuerza bruta para protocolos de las más usadas.

  • Comandos de Hydra:
  • 'hydra -h' - Muestra la ayuda de la herramienta.
  • 'hydra -l -p ssh' - Realiza un ataque de fuerza bruta sobre SSH con un usuario y contraseña especificados.
  • 'hydra -L -P ssh' - Usa listas de usuarios y contraseñas para atacar el servicio SSH.
  • 'hydra -t 4 -L -P http-get' - Ataque de fuerza bruta sobre HTTP GET con múltiples usuarios y contraseñas.
  • 'hydra -V -l -P ftp' - Ataque a FTP mostrando detalles del progreso (modo verboso).
  • 'hydra -s -l -P ' - Especifica un puerto personalizado para el protocolo objetivo.
  • 'hydra -C ' - Usa un archivo con credenciales para un ataque de fuerza bruta (usuario:contraseña).
  • 'hydra -f -l -P ssh' - Realiza el ataque y se detiene después de encontrar la primera contraseña correcta.
  • 'hydra -vV -l -P ' - Modo detallado para ver cada intento y el proceso completo.
  • 'hydra -h -l -P http-post-form' - Realiza un ataque de fuerza bruta en un formulario HTTP POST.
  • 'hydra -t 1 -L -P -o resultados.txt ssh' - Ataque con un solo hilo y guarda los resultados en un archivo de texto.

Metasploit framework details

Metasploit, un framework poderoso y popular de exploits y payloads destinado a hacking ético y pentesting.

  • Instalas Metasploit y lo abres con msfconsole [ sudo apt-get install msfconsole -y ]
  • Buscas un exploit con search y lo seleccionas con use.
  • Configuras un payload con set PAYLOAD
  • Ejecutas el ataque con exploit.
  • Usas módulos de post-explotación para realizar tareas adicionales.
  • Nota: con RHOST seleccionas el host victima [remoto]
  • Nota: con LHOST seleccionas el localhost por la ip [maquina atacante]
  • Nota: con RPORT para puerto vicitima / remoto.
  • Nota: con LPORT es para puerto local de la maquina atacante.

Ncrack tool details

Ncrack es una herramienta de fuerza bruta diseñada para pruebas de acceso y auditoría de contraseñas en varios servicios.

  • Comandos de Ncrack:
  • 'ncrack --help' - Proporciona ayuda sobre el uso de la herramienta.
  • 'ncrack list' - Lista todos los módulos disponibles en el sistema.
  • 'ncrack install' - Instala módulos adicionales, como herramientas para auditar SSH o otros servicios.
  • 'ncrack remove' - Elimina los módulos seleccionados.

Django Framework

Django es un framework de desarrollo web de código abierto y gratuito para Python. Está diseñado para facilitar la creación de aplicaciones web complejas y de alto rendimiento de manera rápida y eficiente.

  • se importa con ' import django '
  • ' from django.shortcuts import render '
  • ' from django.http import HttpResponse '
  • Ex; crear proyecto con django ' django-admin startproject myproject '
  • Ex; Crear una app python dentro de un proyecto ' python manage.py startapp myapp '
  • Ex: Ejemplo django de vista: def my_view(request): return HttpResponse("¡Hola, Mundo!")
  • Ex: Renderizar plantilla HTML [ DJANGO ] def my_view(request): return render(request, 'index.html')
